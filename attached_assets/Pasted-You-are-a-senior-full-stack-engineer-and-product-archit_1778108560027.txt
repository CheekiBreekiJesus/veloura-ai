You are a senior full-stack engineer and product architect.

Build an MVP web application called “AI Identity Stylist” — a personalized AI beauty and fashion assistant that analyzes a user’s selfie and generates a structured aesthetic profile with recommendations.

## CORE GOAL
The app takes a selfie, analyzes facial features using an AI vision model (OpenAI GPT-4o or equivalent), and generates:
- face shape classification
- skin tone + undertone
- style archetype
- beauty + fashion recommendations
- outfit suggestions
- glasses + hairstyle suggestions
- product recommendations (mock affiliate links)

The app must feel like a premium “AI stylist dashboard”, not a generic chatbot.

---

# TECH STACK (MANDATORY)
- Frontend: React (Vite or Next.js acceptable)
- Backend: Node.js (Express)
- Styling: Tailwind CSS
- AI: OpenAI API (vision-capable model like GPT-4o)
- Storage: simple in-memory or local JSON (NO complex infra)
- File upload: basic image upload handling
